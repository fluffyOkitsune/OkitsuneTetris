##おきつねテトリス  
https://fluffyokitsune.github.io/OkitsuneTetris/

# 仕様
* おきつねをテトリミノで潰すと呪われ天罰を受ける  
* おきつねを他のブロックと一緒にラインクリアするとアイテムゲット  
……が特徴のかわいいテトリス

# アイテム
* サポートシャドウ：ハードドロップで落ちる場所を教えてくれる。  
* NEXTボックス：次に出るテトリミノが分かる。　５つまでふえる。　便利。  
* HOLDボックス(A,S,D)：落下中テトリミノを一時的にホールドできる。　３つまでふえる。　便利。  
* ボム：積んだテトリミノを木っ端みじんにする。  
* スピードダウン：テトリミノの落下スピードが１～３段階減少  
* フィーバー：しばらくＩ型のみが降ってくる  
* スコア：うれしい  
* おはらい：おきつねののろいが解ける
* いなり：フィールドのおきつねがいなくなる

# おきつねののろい
* 猟師：NEXTからおきつねがいなくなる  
* 猟犬：スピードアップ  
* 妖狐：消せないつぶせないおきつね「妖狐」が出現。　一定時間で去って行く。  
* 荒天：積んだテトリミノの一部が打ち抜かれて穴が開く  
* 無常：NEXTボックスがなくなる。  
* 呪縛：使いづらいＳorＺ型しばり  
* 奈落：１マス落ちるとハードドロップ  
* 背水：回転が不可能になる  
* 暗黒：少しの時間画面が真っ暗になり、なんも見えなくなる  
* 虚無：一定時間アイテムとスコアが入手できない  
* 黄昏：テトリミノが透明になる  
* 泡沫：スコアが０になる。　４きつね同時潰しで得られる最大の恐怖  
* 終焉：ゲームオーバーになる。　４きつね同時潰しで得られる最恐の恐怖

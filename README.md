## OkitsuneTetris
おきつねテトリス  
https://fluffyokitsune.github.io/OkitsuneTetris/

# 仕様
* おきつねをテトリミノで潰すと呪われ天罰を受ける  
* おきつねを他のブロックと一緒にラインクリアするとアイテムゲット  
……が特徴のかわいいテトリス

# アイテム
* サポートシャドウ：数ターンハードドロップで落ちる場所を教えてくれる。
* NEXTボックス：次に出るテトリミノが分かる。　５つまでふえる。　便利。
* HOLDボックス(A,S,D)：落下中テトリミノを一時的にホールドできる。　３つまでふえる。　便利。
* ボム：積んだテトリミノを木っ端みじんにする。
* スピードダウン：テトリミノの落下スピードが数ターン減少
* フィーバー：しばらくＩ型のみが降ってくる。
* スコアアップ：数ターン取得スコア３倍。うれしい
* おはらい：おきつねののろいが解ける
* いなり：数ターンおきつねのみ出現。　もっふもふ

# おきつねののろい
呪いは全てターン制  
* 猟師：NEXTからおきつねがいなくなる
* 猟犬：数ターン10の間レベル分スピードアップ
* 黄昏：数ターンの間テトリミノが透明になる。　サポートシャドウがないときつい
* 荒天：積んだテトリミノの一部が打ち抜かれて穴が開く。
* 無常：NEXTボックスが見えなくなる
* 呪縛：使いづらいＳorＺ型しばり。　フィーバーを失う
* 奈落：１マス落ちるとハードドロップ
* 背水：数ターンの間、回転が不可能になる。　１ライン消すと解除
* 暗黒：１ターン画面が真っ暗になり、なんも見えなくなる。　２きつね潰しで発生
* 虚無：一定時間アイテムとスコアが入手できない。　２きつね潰しで発生
* 泡沫：入手したスコア・アイテムがなくなる。　４きつね同時潰しで得られる最大の恐怖
* 終焉：ゲームオーバーになる。　４きつね同時潰しで得られる最恐の恐怖

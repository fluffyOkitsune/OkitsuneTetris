# OkitsuneTetris
おきつねテトリス

# 仕様
・おきつねをテトリミノで潰すと呪われ天罰を受ける
・おきつねを他のブロックと一緒にラインクリアするとアイテムゲット
……が特徴のかわいいテトリス

# アイテム
・NEXTボックス：次に出るテトリミノが分かる。　５つまでふえる。　便利。
・HOLDボックス：落下中テトリミノを一時的にホールドできる。　３つまでふえる。　便利。
・ボム：積んだテトリミノを木っ端みじんにする。
・スピードダウン：テトリミノの落下スピードが１〜３段階減少
・フィーバー：しばらくＩ型のみが降ってくる
・スコア：うれしい

# おきつねの天罰
・スピードアップ
・ＳorＺ型しばり
・１マス落ちるとハードドロップ
・回転不可
・闇に覆われ何も見えなくなる
